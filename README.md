# 🚖 Uber Trip Analysis Dashboard (Power BI)

## 📌 Project Overview

Uber Trip Analysis is an end-to-end data analytics project built using Power BI to analyze ride-sharing data and uncover actionable insights. The dashboard provides a comprehensive view of bookings, revenue, trip distance, and customer behavior.

## 🎯 Objectives

* Analyze total bookings and revenue trends
* Understand customer behavior and usage patterns
* Identify peak demand hours and days
* Evaluate vehicle performance
* Discover high-demand pickup and drop-off locations

## 📊 Key Features

* Interactive Power BI dashboard
* KPI cards for quick insights
* Time-based trend analysis
* Location-based insights
* Vehicle type performance analysis
* Payment method distribution

## 📈 Dashboard Sections

### 1. Overview Analysis

* Total Bookings, Revenue, Avg Booking Value
* Trip Distance and Time Metrics
* Payment Type Distribution
* Trip Type (Day/Night) Analysis
* Location Insights

### 2. Time Analysis

* Bookings by Pickup Time
* Day-wise Booking Trends
* Peak Hours Identification
* Heatmap for Hourly Demand

## 🛠️ Tools & Technologies

* Power BI
* Data Visualization
* Data Modeling
* DAX (Data Analysis Expressions)

## 📂 Dataset

The dataset includes trip-level data such as:

* Booking ID
* Pickup & Drop-off Locations
* Trip Distance & Time
* Payment Type
* Vehicle Type
* Booking Value

## 🚀 Key Insights

* Peak bookings occur during evening hours
* Weekends show higher demand compared to weekdays
* UberX contributes the highest number of bookings
* Certain locations dominate pickup and drop-off frequency.
